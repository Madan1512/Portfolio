My Projects

Driver Drowsiness Detection

The UTA Real-Life Drowsiness Dataset is used to train the model. Videos are converted into frames using the Open Computer Vision Library, and frames are annotated using the RoboFlow tool. The YOLOv8 model is used to train and build the Driver Drowsiness Model, which detects whether a driver is drowsy, sleepy, or awake. The model achieves accuracies of 91%, 86%, and 96% for each label, respectively.

Road Traffic Accident Severity Prediction

The Road Traffic Accident (RTA) Dataset is used to train the model. The dataset contains class labels indicating the severity of injuries: slight injury, serious injury, and fatal injury. Numpy, Pandas, and Scikit-Learn libraries are used to import the dataset, perform feature extraction, and train various machine learning algorithms. The algorithms used include Decision Tree Classifier, Random Forest Classifier, AdaBoost Classifier, Support Vector Machine, and Multinomial Naive Bayes. The Support Vector Machine achieved the highest accuracy of 96%.

Weather Prediction

The Seattle Weather Dataset is used to train the model, containing four class labels: sun, rain, drizzle, and snow. Five machine learning algorithms are used for prediction: Decision Tree Classifier, Random Forest Classifier, K-Nearest Neighbor (KNN) Classifier, Logistic Regression, and Extreme Gradient Boosting Classifier (XGBoost). KNN and XGBoost achieved maximum accuracies of 98.6% and 97%, respectively.
